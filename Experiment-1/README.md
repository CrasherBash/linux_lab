**Experiment Aim**:

****To install and run a Linux virtual machine using virtualization software and verify its successful booting.****

**Objective**

To understand the concept of virtualization

To learn how to import and run an OVA file

To verify proper working of a Linux virtual machine

**About virtualisation**

Virtualization is a technology that allows multiple operating systems to run on a single physical machine using software like Oracle VirtualBox or VMware Workstation.
An OVA (Open Virtual Appliance) file contains a pre-configured virtual machine with an operating system and required settings. By importing this file into virtualization software, a complete system can be set up quickly without manual installation.


**Steps to install linux Using VMware**

Step 1: First, VMware Workstation was installed.
Step 2: Then, the application was launched.
Step 3: After that, Open a Virtual Machine was selected.
Step 4: Then, the OVA file was browsed and selected.
Step 5: After that, the appliance was imported and converted.
Step 6: Then, hardware settings were reviewed.
Step 7: After that, the virtual machine was powered on.
Step 8: Finally, login was done and proper booting was verified.


**Results**

Virtual machine imported successfully

Linux operating system booted without errors

System responded properly to commands
