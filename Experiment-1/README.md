**How I installed Linux Using VMware Workstation**

Step 1: First, VMware Workstation Player/Pro was installed on the system.
Step 2: Then, VMware Workstation was launched to proceed further.
Step 3: After that, the option Open a Virtual Machine was selected from the interface.
Step 4: Then, the provided Linux OVA file was browsed and selected.
Step 5: After that, VMware automatically started importing and converting the appliance into a usable virtual machine format.
Step 6: Then, the hardware settings such as memory, processor, and storage were reviewed and modified if required.
Step 7: After that, the virtual machine was powered on.
Step 8: Finally, login was performed and it was confirmed that the Linux system booted successfully and was working properly.
